# Assignment Tracker

A clean, responsive CRUD web application designed to help students track academic assignments by course, due date, and progress status. The application allows users to create, view, update, and delete assignments while keeping everything organized in one place.

## Features

- Create new assignments
- View all assignments in a table format
- Edit existing assignments
- Delete assignments (with confirmation prompt)
- Status tracking (*Not Started, In Progress, Done*)
- Fully responsive UI
